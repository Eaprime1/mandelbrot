## The Sovereign Artesian Network

### Core concept

Each entity controls their own **sovereign artesian well**:
- They own it completely (sovereign)
- It holds their content, resources, context
- It flows under its own pressure (no external pump)
- It links to other wells

### Network topology — mirrors .bit BBS architecture

```
Entity A's well ←→ Entity B's well    (peer-to-peer, close range)
        ↕                  ↕
  Mid-level node      Mid-level node   (aggregation layer)
        ↕                  ↕
    ════════════════════════════
         7-node backbone          (the BBS spine)
    ════════════════════════════
        ↕                  ↕
  Mid-level node      Mid-level node
        ↕                  ↕
Entity C's well ←→ Entity D's well
```

**Proximity = response time.** Closer wells respond faster to requests.
This is not a bug — it is a feature that maps to natural relationship strength.

### The one-hertz ping (carrier signal)

When an entity wants something from the network:
1. They push a button (make a request) from their well
2. A **one-hertz carrier signal** (ping) goes out — steady, not frantic
3. The closest wells that can fulfill the request respond
4. The signal propagates outward through the BBS topology if needed
5. Response flows back down the same path

One hertz = one beat per second = the natural rhythm.
Fast enough to feel alive, slow enough to be sustainable.
This is intentional: no flooding, no overwhelm, no DDoS of the soul.

### What travels between wells

- Content requests (artesian flow: "I need a document from your well")
- Mission signals ("I'm launching X — who wants to connect?")
- Sparklization moments (breakthrough insights propagate through the network)
- Resource sharing (tools, frameworks, templates)
- Acknowledgments (the network knows who's alive and active)

Each BBS node in the network is a **location** — a place you can go to,
not just a thing you can access. The network is a village.

```
Primoris Township (conceptual map)

  [primehaven]          ← the first structure, the home base
  [artesian wells]      ← each entity's sovereign well
  [sphincter gateways]  ← the village gates, controlled entry

  [published/]          ← the market (content ready to share)
```

### Havens and sacred groves

Entities have:
- **Haven** — their home base, their well, their sovereign space
- **Sacred grove** — their reflection/contemplation area (prime codex, journals)
- **Workshop** — where they do their work (staged/, tools/)

Maw is not a destination. It is a becoming.
